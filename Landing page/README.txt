# 2-Page Business Website

A modern and responsive 2-page business website built using HTML, CSS, and JavaScript.

## Features

- Responsive Landing Page
- Professional Business Design
- Request a Quote Form
- Clean User Interface
- Mobile-Friendly Layout
- Simple JavaScript Form Handling

## Pages

### 1. Landing Page
- Hero Section
- Call-to-Action Button
- Services Section
- Navigation Menu
- Footer

### 2. Request a Quote Page
- Full Name Field
- Email Address Field
- Phone Number Field
- Company Name Field
- Service Selection
- Project Details Text Area
- Submit Button

## Technologies Used

- HTML5
